## üìä Project Summary

### Vision
SmartBudget is a modern personal finance web application focused on **simplicity + intelligence**. Unlike feature-heavy competitors (Mint, YNAB), SmartBudget targets users who want basic expense tracking without complexity overload.

---

### MVP Features (8 weeks)
- ‚úÖ User authentication (email/password)
- ‚úÖ Manual transaction entry (6 categories)
- ‚úÖ Monthly budget tracking with alerts
- ‚úÖ Visual dashboard with spending breakdown
- ‚úÖ Transaction history with filtering

---

### Tech Stack
- **Frontend:** React 18 + TypeScript + Material-UI + Recharts
- **Backend:** Node.js + Express + TypeScript
- **Database:** PostgreSQL 15
- **Deployment:** Heroku + GitHub Actions CI/CD
- **Testing:** Jest, Supertest, Playwright

---

### Success Metrics
| Metric | Target |
|--------|--------|
| Users (Month 3) | 1,000 |
| Retention (30 days) | 60% |
| Transaction entry time | < 10 seconds |
| Backend test coverage | 80%+ |

---

### Timeline
- **Sprint 1 (Weeks 1-2):** Foundation & Auth
- **Sprint 2 (Weeks 3-4):** Transaction Management
- **Sprint 3 (Weeks 5-6):** Dashboard & Budgets
- **Sprint 4 (Weeks 7-8):** Polish & Launch
